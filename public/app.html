Setup Guide
1. Download & Install
Run the installer. Windows may show a SmartScreen warning — click "More info" then "Run anyway". We're working on code signing.
2. Sign In
Use your OptionsLab mobile number and MPIN to log in to Jobber Pro. Same credentials you use here.
3. Connect Angel One
Enter your Angel One Client ID, API Key, and MPIN inside the app. Your credentials are stored locally — we never receive them.
